# Denon AVR

A minimal wrapper around the Denon AVR control protocol (link) (tested on fdjk*). Provides an easy way of interfacing with Denon devices over a network while handling errors and corrupted data gracefully. Use as a lib or check out an example TUI demo built with Ratatui.
